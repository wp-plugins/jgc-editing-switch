A simple plugin allows you to quickly change page or post in the edit screen while editing.

== Description ==

The plugin adds a metabox on screen editing pages and posts to quickly change page or post.

The metabox page contains pages with status 'Published', 'Scheduled', 'Draft', 'Pending Review' and 'Private'.

The metabox post contains posts with status 'Scheduled', 'Draft', 'Pending Review' and 'Private'.

== Installation ==

1. Unzip the file 'jgc-editing-switch.zip'
2. Upload the 'jgc-editing-switch' folder to the '/wp-content/plugins/' directory

3. Activate the plugin through the 'Plugins' menu in WordPress
